# What-a-Div

A small browser-based "What-a-Div" (Whack-a-Div-style) game â€” click the active squares to score points before time runs out.

How to run
1. Clone the repo or download the files.
2. Open `index.html` in your browser.

Controls and options
- Click the highlighted square to score.
- Game runs for a preset time (see `script.js`).
- You can customize grid size, game duration, and intervals in `script.js`.
